# 💳 UPI Payments Performance & Risk Analytics Dashboard

## 📌 Project Overview

The **UPI Payments Performance & Risk Analytics Dashboard** is an interactive Business Intelligence solution developed in **Microsoft Power BI** to analyze UPI transaction performance, payment trends, fraud exposure, and banking insights.

This project demonstrates end-to-end data analytics by transforming raw transactional data into meaningful business insights using **Power Query**, **DAX**, and **interactive dashboards**.

The dashboard enables stakeholders to monitor key performance indicators (KPIs), identify transaction patterns, analyze fraud rates, and support data-driven decision-making.

---

## 🎯 Business Objective

The objective of this project is to:

- Monitor overall UPI transaction performance.
- Analyze monthly transaction trends.
- Track total transaction value and average transaction amount.
- Measure fraud transactions and fraud rate.
- Analyze transaction success and failure rates.
- Compare transaction value across banks and spending categories.
- Provide interactive dashboards for business reporting.

# 📊 Dashboard Features

## Executive Dashboard

Provides a high-level overview of the complete UPI ecosystem through interactive KPI cards.

### Included KPIs

- Total Transactions
- Total Transaction Value
- Average Transaction Value
- Fraud Transactions
- Fraud Rate
- Failed Transactions

---

## Monthly Transaction Trend

Analyzes transaction value across different months to identify business growth and seasonal trends.

---

## Transaction Value by Sender State

Visualizes transaction value contributed by different states to identify regional transaction patterns.

---

## Transaction Value by Time Bucket

Categorizes transactions into:

- Morning
- Afternoon
- Evening
- Night

to understand customer payment behavior throughout the day.

---

## Category-wise Transaction Analysis

Compares transaction values across spending categories including:

- Shopping
- Grocery
- Utilities
- Fuel
- Food
- Entertainment
- Travel
- Transport

---

## Receiver Bank Analysis

Analyzes transaction value distribution among major receiving banks.

Example banks include:

- Axis Bank
- HDFC Bank
- ICICI Bank
- SBI
- PNB
- Yes Bank
- IndusInd Bank
- Kotak Mahindra Bank

---

## Transaction Status

Compares:

- Successful Transactions
- Failed Transactions

to monitor transaction reliability.
